## 📖 Overview

The **Math Game Console Application** is an elegant and professional educational
tool designed to help users practice fundamental arithmetic operations. Built
with C# and featuring a clean console interface, this application provides an
engaging way to strengthen mathematical skills through interactive gameplay with
comprehensive progress tracking.

### ✨ Key Highlights

- **🎯 Four Core Operations**: Addition, Subtraction, Multiplication, and
  Division
- **🎮 Interactive Gameplay**: User-friendly menu system with flexible question
  counts
- **📊 Progress Tracking**: Comprehensive game history with detailed statistics
- **🛡️ Robust Design**: Professional error handling and input validation
- **🎨 Clean Interface**: Polished console UI with intuitive navigation
